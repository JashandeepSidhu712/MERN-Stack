# REACT INSTALLATION

## STEP 1. Open VS code

## STEP 2. Enter folder/directory where you want to create the react-folder
use **cd dropbox**, to enter the dropbox folder.

## STEP 3 npm install -g create-react-app

## STEP 4. npx create-react-app appname

## STEP 5. cd appname

## STEP 6. npm start
